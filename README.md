# Climate & Disasters Data Analysis

Data analytics project focused on climate-related natural disasters, providing country-level insights and interactive geographic visualizations.

[![Live Map](https://img.shields.io/badge/GitHub_Pages-Live_Map-brightgreen?style=for-the-badge)](https://captaina10.github.io/Climate-Disasters-Data-Analysis/outputs/maps/carte_catastrophes_pays.html)

## Project Overview

This project explores climate and natural disaster data to analyze spatial distributions and patterns across countries.
The main objective is to transform raw datasets into clear, interactive visual outputs suitable for web publication and analytical exploration.

The final deliverable is an interactive HTML map deployed via GitHub Pages.

## Installation and Local Execution

Clone the repository:
git clone https://github.com/CaptainA10/Climate-Disasters-Data-Analysis.git

Move into the project directory:
cd Climate-Disasters-Data-Analysis

Install dependencies:
pip install -r requirements.txt

Launch Jupyter Notebook:
jupyter notebook

Run the notebooks located in the notebooks directory to reproduce the analysis and regenerate the interactive map.

## Deployment

The interactive map is deployed using GitHub Pages.

Deployment configuration:
- Branch: main
- Source path: outputs/maps/carte_catastrophes_pays.html

Any update pushed to the repository automatically updates the live map.
